About This Project

This is a school project built as part of an API integration assignment. It uses the Remotive API from the Remotive API to fetch real remote job listings and display them with filters. It is built using only HTML, CSS, and JavaScript.

Remote First

We focus on remote and flexible jobs so you can work from anywhere.

Real-Time Data

Jobs are fetched live from the Remotive API, providing up-to-date remote job listings each time you search.

Always Free

No subscriptions, no hidden fees. Just search and apply.
